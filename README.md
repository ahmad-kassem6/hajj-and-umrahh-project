# Hajj and Umrah Project

## Overview
This is a **frontend project** for Hajj and Umrah services, built with **React** and **Tailwind CSS**. The project provides users with a simple and interactive interface to explore Hajj and Umrah information, plans, and related services.

---

## Features
- Responsive user interface using Tailwind CSS
- React-based dynamic components
- Pages for Hajj and Umrah information
- Easy navigation and clean layout
- Optimized for both desktop and mobile devices
